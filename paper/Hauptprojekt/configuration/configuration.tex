\abstractDE{
Diese Arbeit dokumentiert den Bau eines Prototypen, der es ermöglicht die Geschwindigkeit von Fahrzeugen in Videos zu bestimmen.
Dabei wird ein Faltungsnetzwerk namens YOLO benutzt, um Kennzeichen zu finden anhand deren bekannter Dimensionen über mehrere Bilder hinweg die Geschwindigkeit des Fahrzeugs errechnet werden kann.
}

\abstractEN{
In this work the problem of building a prototype system to estimate the speed of vehicles in a video feed is tackled.
Inspired by the versatility of deep neural networks the system uses a CNN called YOLO to detect license plates and uses their known dimensions to calculate the speed of the vehicle.
}
